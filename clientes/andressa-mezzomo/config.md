## 4. Paleta de Cores

| Nome | Código | Uso |
|------|--------|-----|
| Dark Base | #050D32 | Fundo principal, textos, elementos estruturais |
| Brand Color | #AB8D5F | Elemento de destaque, assinaturas, detalhes premium |
| Light Accent | #AAACAD | Elementos secundários, divisores, ênfases neutras |
| Light Base | #f7f7f7 | Fundo claro, espaço em branco, modernidade |

### Filosofia Cromática
- **Dark Base (#050D32):** Sofisticação e profundidade
- **Brand Color (#AB8D5F):** Ouro arenado que evoca exclusividade, luxo sutil e confiança refinada (não ouro puro, é refinado e discreto)
- **Light Accent (#AAACAD):** Equilíbrio neutro e elegância
- **Light Base (#f7f7f7):** Clareza, modernidade e espaço respirável
